# claims-iq-pro-agentic-pnc
ClaimsIQ Pro is a multi-agent P&amp;C insurance claims processor bridging a React frontend with an n8n &amp; OpenAI (GPT-4o) backend via webhooks.  Intake Agent: Normalizes messy text fields.  Coverage Agent: Validates active policy boundaries.  Fraud Agent: Screens text for high-risk anomalies.
